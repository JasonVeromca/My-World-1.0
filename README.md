##   2.模块拆分  ##

   >管理员模块：管理员拥有接收和通过登录系统查询用户使用记录、商品交易记录等功能
   >用户模块：分为普通用户和特殊用户
   >收付款模块：特殊用户采用刷指定卡付款，普通用户支持现金、微信、支付宝，支持找零
   >商品模块：不同商品对应不同商品指示灯和不同选择键，某商品断货情况下可以自动向管理者系统发出补货通知
